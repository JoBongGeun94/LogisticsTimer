

# 물류 작업현장 Gage R&R 분석 시스템

**MSA 규격 완전 준수 · SOLID 원칙 적용 · 실시간 분석 · 다크모드 지원**

공군 종합보급창의 물류 작업현장에서 사용하는 정밀한 인시수 측정 및 Gage R&R 분석 도구입니다.

## ✨ 주요 특징

### 🎯 MSA 규격 완전 준수
- **최소 측정 요구사항**: 6회 측정, 2명 이상 측정자, 5개 이상 대상자
- **실시간 검증**: 세션 생성 시 MSA 요구사항 자동 검증
- **ANOVA 분산 분석**: 통계적 유의성 검증
- **P/T 비율 분석**: 측정시스템 정밀도 평가

### 📊 실시간 고급 분석
- **즉시 분석**: 측정과 동시에 실시간 통계 분석
- **핵심 지표**: Gage R&R, ICC, ΔPair, 변동계수(CV) 실시간 표시
- **상태 평가**: 우수/양호/보통/불량 자동 판정
- **상세 분석 모달**: 분산 구성요소, 해석 및 권장사항 제공

### 🌙 최적화된 사용자 경험
- **모바일 최적화**: 터치 친화적 인터페이스, 반응형 레이아웃
- **다크모드 지원**: 현장 환경에 최적화된 테마 (기본값)
- **키보드 단축키**: 스페이스바(시작/정지), Enter(측정완료), R(리셋), F1(도움말)
- **토스트 알림**: 실시간 피드백 및 상태 표시
- **뒤로가기 보호**: 데이터 손실 방지 기능

### 🏗️ 견고한 아키텍처
- **SOLID 원칙**: 단일 책임, 개방-폐쇄, 의존성 역전 적용
- **TypeScript 100%**: 완전한 타입 안전성
- **메모이제이션**: React.memo, useMemo 적용으로 성능 최적화
- **에러 경계**: 예외 처리 및 복구 메커니즘

## 📊 분석 기능

### Gage R&R 분석
```typescript
// 실시간 분석 결과 예시
{
  gageRRPercent: 15.2,    // Gage R&R 비율
  repeatability: 0.0045,  // 반복성
  reproducibility: 0.0032, // 재현성
  iccValue: 0.85,         // 급내상관계수
  deltaPair: 0.025,       // 연속 측정 차이
  cvPercent: 8.9,         // 변동계수
  status: 'acceptable'    // 평가 상태
}
```

### 평가 기준
| 구분 | R&R 비율 | ICC | CV | 평가 |
|------|----------|-----|-----|------|
| 우수 | < 10% | ≥ 0.9 | ≤ 12% | 모든 측정 권장 |
| 양호 | 10-30% | 0.7-0.9 | 12-20% | 대부분 사용 가능 |
| 보통 | 30-50% | 0.5-0.7 | 20-30% | 제한적 사용 |
| 불량 | > 50% | < 0.5 | > 30% | 즉시 개선 필요 |

## 📖 사용법

### 1. 세션 생성
```
새 세션 → 세션명 입력 → 작업 유형 선택 → 측정자/대상자 설정 → 생성
```
- **MSA 권장**: 측정자 2명 이상, 대상자 5개 이상
- **실시간 검증**: 요구사항 미달 시 경고 메시지 표시

### 2. 측정 진행
```
측정자/대상자 선택 → 스페이스바(시작) → Enter(완료) → 자동 기록
```
- **정밀도**: 0.01초 단위 측정
- **키보드 지원**: 마우스 없이도 완전 조작 가능

### 3. 실시간 분석 확인
```
측정 횟수 → 평균 시간 → 변동계수 → Gage R&R → ICC → ΔPair
```
- **6회 이상**: 완전한 Gage R&R 분석 가능
- **실시간 상태**: 색상 코딩으로 즉시 판단

### 4. 데이터 활용
```
CSV(기본 데이터) → 분석(Excel 보고서) → 상세(모달 분석)
```
- **필터링**: 특정 측정자/대상자만 선별 분석
- **관리**: 개별 기록 삭제, 전체 초기화
